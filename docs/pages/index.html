Introduction

ZUI is a pattern library developed for Zource. This library is a collection of commont user interface elements that are used in advanced web applications. Zource makes use of these elements. It is important to realize that ZUI is not designed for your average daily website. It's meant to be used in web-applications.

Why this library

This library has been developed to make sure that the visual appearance and the functionality of the user interface works the same throughout applications made by Zource. A smooth user experience for Zource applications is important and ZUI plays a big part to accomplish this.

Acknowledgement

This library is developed from the ground up. Yet we did use existing libraries as reference material. We especially love Atlassian's AUI and Zurb's Foundation. This library also makes extensive use of jQuery.
