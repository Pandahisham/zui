Summary

A flexbox grid system which makes it possible to create responsive pages.

  • HTML Attributes
  • HTML Classes
  • JavaScript Events
  • JavaScript Methods
  • SASS Functions
  • SASS Mixins
  • SASS Variables

There are no attributes defined for this pattern.

There are no classes defined for this pattern.

There are no events defined for this pattern.

There are no methods defined for this pattern.

There are no functions defined for this pattern.

There are no mixins defined for this pattern.

Name Description
flexgrid-class-container The name of the class used to define a flexgrid container.
flexgrid-class-container-fluid The name of the class used to define a fluid flexgrid container.
flexgrid-class-row The name of the class used to define a row in a flexgrid container.
flexgrid-class-row-reverse The name of the class used to define a row in a flexgrid container.
flexgrid-class-xs-column The prefix of the class used to define an extra small size column in a flexgrid container.
flexgrid-class-xs-push The prefix of the class used to push an element in an extra small size column in a flexgrid container.
flexgrid-class-sm-column The prefix of the class used to define a small size column in a flexgrid container.
flexgrid-class-sm-push The prefix of the class used to push an element in a small size column in a flexgrid container.
flexgrid-class-md-column The prefix of the class used to define a medium size column in a flexgrid container.
flexgrid-class-md-push The prefix of the class used to push an element in a mediumn size column in a flexgrid container.
flexgrid-class-lg-column The prefix of the class used to define a large size column in a flexgrid container.
flexgrid-class-lg-push The prefix of the class used to push an element in a large size column in a flexgrid container.
flexgrid-column-count The amount of columns to use in the grid.
flexgrid-media-query-xs The media query used to match extra small screens.
flexgrid-media-query-sm The media query used to match small screens.
flexgrid-media-query-md The media query used to match medium screens.
flexgrid-media-query-lg The media query used to match large screens.
