Summary

The steps pattern should be used to visualize the steps in a wizard.

  • HTML Attributes
  • HTML Classes
  • JavaScript Events
  • JavaScript Methods
  • SASS Functions
  • SASS Mixins
  • SASS Variables

There are no attributes defined for this pattern.

Name Description
zui-steps The zui-steps turns an ordered list into a steps overview.
zui-steps-step The zui-steps-step class should be set on a list item element. It tells the pattern that the list item is a single step and makes the step active by default. All steps that come after the zui-steps-current will be inactive.
zui-steps-step-current The zui-steps-step-current class is a trigger to tell the pattern that all zui-steps-step that come after this element will be inactive.

There are no events defined for this pattern.

There are no methods defined for this pattern.

There are no functions defined for this pattern.

There are no mixins defined for this pattern.

There are no variables defined for this pattern.
