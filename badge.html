Summary

Badges are meant to provide a visual identification of numeric values. They are not used for anything other than integers. For textual representations, use the status pattern.

HTML Attributes

There are no attributes defined for this pattern.

HTML Classes

Name Description
zui-badge The zui-badge class identifies an HTML element as a badge. It is recommended that you use this class on a span element.

JavaScript Events

There are no events defined for this pattern.

JavaScript Methods

There are no methods defined for this pattern.

Sass Functions

There are no functions defined for this pattern.

Sass Mixins

There are no mixins defined for this pattern.

Sass Variables

Name Description
badge-background The background color of a badge.
badge-color The text color of a badge.
