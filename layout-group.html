Summary

The layout group pattern is used to define sections in HTML. It is a simple form of a grid.

  • HTML Attributes
  • HTML Classes
  • JavaScript Events
  • JavaScript Methods
  • SASS Functions
  • SASS Mixins
  • SASS Variables

There are no attributes defined for this pattern.

Name Description
zui-layout-group The class that describes a row of elements that have the class zui-layout-item.
zui-layout-item The class that describes a cell within an element that has the zui-layout-group class.

There are no events defined for this pattern.

There are no methods defined for this pattern.

There are no functions defined for this pattern.

There are no mixins defined for this pattern.

There are no variables defined for this pattern.
